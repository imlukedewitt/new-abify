# Development Guide: Main

## Prerequisites
- Ruby
- Bundler

## Installation
1. Navigate to the project root directory.
2. Run `bundle install` to install Ruby dependencies.

## Environment Setup
- Configuration files are located in the `config/` directory.
- Environment variables are managed via `.env` files (e.g., `.posting/.env`).

## Local Development Commands
- To start the development server, run `rails server` (inferred from `Procfile.dev`).

## Build Process
- Build and task automation is managed through `rake` commands (from `Rakefile`).

## Testing Strategy
- The project uses RSpec for testing.
- Tests are located in the `spec/` directory.
- To run tests, use the `rspec` command.
